QuitoEmprende
QuitoEmprende es una plataforma web desarrollada con JavaScript y React que permite a emprendedores en Quito subir, promocionar y gestionar sus productos de manera fácil y accesible. El proyecto busca impulsar el comercio local ofreciendo un espacio digital donde los emprendedores puedan visibilizar sus ofertas sin complicaciones técnicas.

El frontend está construido en React para una experiencia de usuario dinámica y responsiva, mientras que el backend provee las APIs necesarias para la gestión de productos y usuarios.

El proyecto está desplegado en la plataforma Railway, garantizando un hosting escalable y confiable.
